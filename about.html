Why I Built This !

V-Planner was developed as a personal learning project to gain hands-on experience with modern web technologies. By combining form inputs, table manipulation, real-time validation, and dynamic PDF generation, this project serves as a practical example of building a complete front-end application.

I created this mini software with the goal of mastering:

  • DOM manipulation and event handling in JavaScript.
  • User interface feedback with libraries like SweetAlert.
  • Generating reports directly from client-side data.
  • Structuring responsive pages with Bootstrap.
